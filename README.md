# IBM Contract Risk manager

Contract Risk manager is a contract risk review prototype built to demonstrate how enterprise AI can support faster legal and commercial decision-making.

The application provides a structured dashboard for reviewing contract language, surfacing clause-level risk, highlighting missing protections, and generating negotiation guidance and executive-ready summaries.

This project uses:

- **Next.js** for the frontend and backend route handling
- **Vercel** for deployment
- **IBM watsonx.ai** for live AI-powered contract analysis

The current prototype supports live analysis from pasted contract text. Document and image ingestion are presented as future workflow extensions within the product experience.

## Tech Stack

- Next.js
- TypeScript
- Vercel
- IBM watsonx.ai

## Notes

This project was created as an AI-focused proof of concept and is designed to showcase:
- a clean enterprise-style interface
- full-stack integration with IBM watsonx
- structured AI outputs for business use cases
